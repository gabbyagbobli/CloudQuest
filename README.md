# 🌌 CloudQuest: The Google Cloud AI Odyssey

[![Live Platform](https://img.shields.io/badge/Live-CloudQuest-blue?style=for-the-badge&logo=google-cloud)](https://cloudquest-platform-12345.web.app)
[![Tech Stack](https://img.shields.io/badge/Stack-Next.js%20|%20Firebase%20|%20Framer%20Motion-black?style=for-the-badge)](https://nextjs.org)

**CloudQuest** is a premium, gamified educational platform designed to take beginners from "Ground Control" to "Fleet Commander" in the Google Cloud AI ecosystem. Built with a cinematic space-exploration narrative, it transforms complex cloud concepts into an immersive mission-based journey.

---

## 🚀 The Curriculum

The platform is structured into four distinct "Nebulas," each representing a core pillar of the Google Cloud AI stack:

### 📡 Level 1: Launch Sequence (Foundations)
*   **The Blueprint**: Regions, Zones, and Resource Hierarchy.
*   **The Command Chain**: IAM, Service Accounts, and Security.
*   **The Engines**: Compute Engine, Cloud Run, GKE, and Serverless.

### ☄️ Level 2: The Data Nebula (Data & Storage)
*   **The Vault**: Cloud Storage and Storage Classes.
*   **The Library**: BigQuery, BigQuery ML, and Dataplex.
*   **The Courier**: Pub/Sub, Dataflow, and real-time ingestion.

### 🧠 Level 3: The Gemini Core (Generative AI)
*   **The Engine**: Large Language Models (LLMs) and Multimodality.
*   **The Science**: Tokens, Context Windows, and Temperature.
*   **The Pilot**: Prompt Engineering, Chain of Thought, and Safety Filters.

### 🤖 Level 4: Gemini Agent Platform (Enterprise Agents)
*   **The Fleet**: Autonomous Agents and Reasoning Engines.
*   **The Anchor**: Grounding with Google Search and Enterprise Data.
*   **The Armor**: Model Armor, ADK, and Multi-Agent Orchestration.

---

## ✨ Features

*   **Cinematic UI/UX**: A high-fidelity "Glassmorphism" design system with smooth Framer Motion transitions and space-themed aesthetics.
*   **Deep-Dive Masterclass**: Over 60 detailed lesson steps across the curriculum (15 per level).
*   **Multi-Stage Exams**: Rigorous 5-question assessments for every level to verify "Pilot Competency."
*   **Real-time Gamification**: Integrated with Firebase for user authentication, progress persistence, and a global "Aura" (XP) Leaderboard.
*   **GDE Ready**: Designed as a portfolio piece for the Google Developer Expert application, focusing on technical depth and educational impact.

---

## 🛠️ Tech Stack

*   **Frontend**: Next.js 16 (App Router), TypeScript, Framer Motion.
*   **Styling**: Vanilla CSS with custom design tokens for glassmorphism and neon accents.
*   **Backend**: Firebase (Authentication, Firestore, Hosting).
*   **Icons**: Lucide React.
*   **Deployment**: Automated via Firebase CLI with static export optimization.
